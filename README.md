# JNU-NLP 学习路径

欢迎你加入到暨南大学NLP课题组的前期培训当中！本次课题组培训周期为66天，从即日起到2023年9月1日，你将进行如下内容的学习和培训、掌握后续科研工作相关的知识以及技术，并在最终完成学习考核。通过考核的成员将会按照个人兴趣被分配到各个研究课题方向当中，以进行后续的科研工作。
